# Checklist

## Before CERN registration

- [ ] Confirm your CERN category with your supervisor or team leader.
- [ ] Confirm your home institute paperwork.
- [ ] Know your SHiP contact person.
- [ ] Know whether you need physical site access.
- [ ] Know whether your work requires safety training or dosimetry.

## Account setup

- [ ] CERN account active.
- [ ] Single Sign-On works.
- [ ] Two-factor authentication configured if required.
- [ ] Users Portal profile reviewed.
- [ ] Required groups or mailing lists requested.
- [ ] CERNBox access checked.
- [ ] GitLab or GitHub access checked.
- [ ] Indico access checked.
- [ ] Mattermost or collaboration chat access checked.

## SHiP setup

- [ ] Read the SHiP public overview.
- [ ] Find the relevant SHiP meetings.
- [ ] Find the relevant software repository.
- [ ] Read the matching SHiP software documentation page.
- [ ] Reproduce one documented setup or example.
- [ ] Write down the commands and versions used.

## Computing setup

- [ ] Know whether to use LXPLUS, local machine, or container.
- [ ] Know where input data lives.
- [ ] Know where output should go.
- [ ] Know whether batch jobs are needed.
- [ ] Know who approves access to protected datasets or repositories.

## Working rhythm

- [ ] Regular supervisor meeting scheduled.
- [ ] Working-group meeting identified.
- [ ] First concrete task agreed.
- [ ] First deliverable date agreed.
- [ ] Help path known for administrative, computing, and SHiP-specific questions.
