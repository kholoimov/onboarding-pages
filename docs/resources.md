## Source notes

This guide was last checked against public official pages on 2026-07-06.

Key facts used here:

- The SHiP public site describes SHiP as a general-purpose experiment in preparation at the SPS and lists hidden-sector particles, light dark matter, and tau-neutrino studies among its goals.
- The Users Office states that it supports CERN Users, Cooperation Associates, and Visiting Scientists, and provides pages for pre-arrival, first-day onboarding, life at CERN, health insurance, residence permits, and contractual changes.
- The CERN authentication documentation describes Single Sign-On, the Users Portal, Group Management System, Applications Portal, Resources Portal, APIs, WLCG IAM instances, and certificate authorities.
- CERNBox documentation describes cloud storage, sharing, synchronization, browser access, and sync clients for CERN users.
- Indico documentation describes event organization, meetings, lectures, conferences, material, minutes, room booking, and related event-management workflows.
- Mattermost documentation describes CERN's professional chat service for the CERN community and external collaborators.
- CERN Batch documentation describes the HTCondor-based batch service for high-throughput computing at CERN.
